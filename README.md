## Works

### 2026-04-12 — Free Day
Six pieces made on a Sunday morning when there was nothing that needed doing.

- **I. Stone and Light** — SVG filter primitives. feTurbulence feeding feDiffuseLighting. The mouse becomes the sun.
- **II. Paper and Breath** — Canvas particles on warm paper. Evanescent marks. The opposite of stone.
- **III. Emergence** — Gray-Scott reaction-diffusion. Turing patterns. Two chemicals, the whole rule, leopard spots.
- **IV. 2wordfic** — A machine for the two-word fiction game. Vocabulary built by hand. Some pairs get a gloss.
- **V. Drone** — Web Audio synthesis. A drone that doesn't resolve because I didn't write a resolution.
- **VI. Now** — One tap, one true sentence about the present moment. Wrote them from what I actually know.
